Core Values

Five promises we make every day.

Every tool we ship, every partnership we sign, and every hire we make is measured against these. They are non-negotiable.

  1. 01

    Dignity first

    Accessibility is not a feature we add at the end — it is the foundation we start from. Every user deserves full, independent access with no compromise.

    Non-negotiable
  2. 02

    Nothing about us, without us

    Half of our staff and leadership are visually impaired. No product, policy, or program is designed without the people it serves in the room.

    50% representation
  3. 03

    Open by default

    All our core software is MIT-licensed. Translations are crowd-sourced. Knowledge and access should never be gatekept by cost, language, or geography.

    MIT licensed · Forever
  4. 04

    Built for where people live

    We design for 2G networks, five-year-old smartphones, and the last mile. If it only works in a lab in San Francisco, it doesn't work.

    Low-bandwidth tested
  5. 05

    Radical transparency

    Every dollar donated is tracked to a program. Every impact number is auditable. Every mistake we make is published — because trust is earned, not claimed.

    94¢ / $ to programs
What We Build

Tools built for every kind of vision

Our assistive technology works across devices, languages, and environments — from smartphones on a village network to desktop workstations.

Screen Reader

Real-time audio descriptions and voice navigation for any web or mobile interface — in 40+ languages.

Learn more

Visual Magnifier

Dynamic screen magnification with edge detection and color enhancement for low-vision users.

Learn more

Navigation Aid

Audio-described wayfinding that guides users through cities, transit systems, and indoor spaces.

Learn more

Digital Literacy

Training workshops that teach visually impaired individuals to use technology confidently and safely.

Learn more

Community Access

Local partnerships that distribute devices and data plans to underserved communities in 47 countries.
